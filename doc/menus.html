Navigation menus

Global

Soft CSS comes with 2 types of intuitive menus:

  • - Top navigation bar
  • - Side navigation menu
Both were developped while thinking about responsive design, so they change aspect to fit to all screen sizes.

Classes

  • - nav.top: By putting the class "top" on the nav, you define the menu as a top navbar
  • - div.header and div.footer: These are used to define the header and footer of a side navigation menu ( it does nothing when used on a top navigation bar )
  • - div.toggle: Put a div with the class "toggle" at the beginning of your nav structure. It will act as the toggle button to display the menu on narrow displays, but it won't appear on wide displays
  • - div.links: This will be the container for your links, so put them here
  • - div.submenu: Instead of putting a link, you can put a dropdown submenu containing multiple links by using a div with the class "submenu"
    You need to put an attribute named data-toggle ( either "click" or "hover" ) to define which type of event will trigger the dropdown
  • - div.submenu span: If you want to give a name to your submenu, put it in a span, and then put the div.dropdown after it
  • - div.dropdown: When using a submenu, this div will act as the dropdown containing the link

Top Navbar

Structure

Here is the basic HTML structure for a top navbar: 
            
<nav class="top" data-follow>
  <div class="toggle right"></div>
  <div class="links">
    <div class="submenu" data-toggle="click">
      <span>
        Submenu title
      </span>
      <div class="dropdown">
        <a href="target1">Submenu link 1</a>
        <a href="target2">Submenu link 2</a>
        <a href="target1">Submenu link 3</a>
      </div>
    </div>
    <a href="target4">Normal link</a>
    <a href="target5">Normal link</a>
  </div>
>/nav>
