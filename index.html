Grid Layout

With Soft CSS, you can organize your components using a responsive grid layout. A lot Pretty much all other CSS frameworks can do that, but Soft CSS is different because it's easy to use, and you have full control over it.
The layout is based on a 12-tiles grid, in which you put columns which take a certain number of tiles. For example you can put 4 columns of 3 tiles each. If your viewport's width is coming under the threshold, the columns will stack on top of each other, becoming rows.
